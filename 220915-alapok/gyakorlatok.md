
`alapok-kod-dedukcio`

1. Mit kell a megjelölt részhez írni, hogy a várt eredményt kapjuk?

   ```js
   "hello" + ?       // "helloworld"
   "hello " + ?      // "hello world"
   "hello" + ? + ?   // "hello world"
   ```

1. Mit kell a megjelölt részhez írni, hogy a várt eredményt kapjuk?

   ```js
   ? + 2     // 5
   ? + 2     // "32"
   ? + ? + 2 // "5"
   ```

1. Mit kell a megjelölt részhez írni, hogy a várt eredményt kapjuk?

   ```js
   "legjobb zenekar a " + ?     // "legjobb zenekar a 3 + 2"
   "legjobb zenekar a " + ? + ? // "legjobb zenekar a 32"
   "legjobb zenekar a " + ?     // "legjobb zenekar a 5"
   "legjobb zenekar a " + ? + ? // "legjobb zenekar a 5"
   ```
